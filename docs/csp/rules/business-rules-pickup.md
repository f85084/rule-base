# 商業規則：超商取貨

來源：`data/sources/csp/商業邏輯.pdf`（Redmine #33071）

## 5,000 元以上的付款與配送組合

以下規則以訂單金額是否超過 5,000 元、付款方式及配送方式判斷：

| 超過 5,000 元 | 付款方式 | 配送方式 | 提示 | 結果 |
|---|---|---|---|---|
| 是 | COD | 宅配 | 不提示 | 可送出 |
| 是 | COD | 超取 | 不提示 | 可送出 |
| 是 | ATM | 宅配 | 提示需修改付款方式才能超取 | 可送出修改後的宅配資料 |
| 是 | ATM | 超取 | — | 不可選擇 |
| 否（含 5,000 元） | COD | 宅配 | 不提示 | 可送出 |
| 否（含 5,000 元） | COD | 超取 | 不提示 | 可送出 |
| 否（含 5,000 元） | ATM | 宅配 | 不提示 | 可送出 |
| 否（含 5,000 元） | ATM | 超取 | 不提示 | 可送出 |

## 常溫超商取貨的付款限制

- 商品金額 5,000 元以內（含）：開放其他可支付的非 COD 付款方式，例如禮券、提貨券、現金帳戶、東森幣、信用卡、LINE Pay、台灣 Pay。
- 商品金額超過 5,000 元至 20,000 元（含）：僅限 COD 加東森幣付款。
- 商品金額超過 20,000 元：不可使用超商取貨。
