# 音檔：索引與檔案建立時間落差

來源：`data/sources/csp/音檔相關.pdf`

## 案例

| 項目 | 值 |
|---|---|
| ConnectionId | `01a903a7d5a95b9a` |
| 音檔 | `01a903a7d5a95b9a_0937051087_280239313.wav` |
| 索引建立時間（`create_on`） | 2026-03-16 13:53:52 |
| 音檔建立時間 | 2026-03-16 14:36:23 |
| 時間差 | 約 42 分鐘 |

NAS 路徑：

```text
\\10.33.201.23\RecordFiles\Recording\41188\2026-03-16\
```

## 現象與推論

索引先建立，但實際音檔尚未生成到 NAS。這段時間使用者雖能取得音檔路徑，播放時仍可能因檔案不存在而暫時失敗；待音檔生成後，播放會恢復正常。

排查時應同時比對索引建立時間、音檔實際建立時間及 NAS 檔案是否存在，避免將此類暫時性問題誤判為播放權限或路徑錯誤。
