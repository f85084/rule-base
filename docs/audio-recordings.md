## Lab 調聽流程摘要

1. 使用 Lab 共用 RDS 帳號連線至 `cpub-EHS`。
2. 從 MI 加入要播放的項次。
3. 重新整理，等待音檔出現。
4. 如需驗證或排查，遠端至相關主機確認 API 位置與音檔狀態。

原始文件包含 RDS 與主機登入密碼，未複製到本文件；請從受控的密碼管理或原始文件取得。
